D How to paste this into EmailJS
  1. Go to emailjs.com → Email Templates → Create New Template
  2. Fill in To Email, From Name, Reply To, CC, and Subject from section A above
  3. Click the Content tab → switch to HTML editor (toggle in top-right)
  4. Delete any existing content and paste the raw HTML from section B
  5. Click Save — copy the Template ID (e.g. template_abc1234)
  6. Add the Template ID + your Public Key to .env in the project root
  7. Restart npm run dev — done!
